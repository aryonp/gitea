# Gitea Automator

As its name, automating everything that you need to do to install Gitea

## Requirements

Tested on Ubuntu 16.04

### Installation

1. git clone https://github.com/aryonp/gitea.git gitea-install
2. cd gitea-install/
3. sudo chmod +x *.sh
4. ./install.sh
